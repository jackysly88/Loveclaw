# LoveClaw - 八字缘分匹配

智能八字匹配技能，寻找有缘人

## 功能概述

- 🤖 **智能引导**：对话式收集用户信息
- 🔮 **八字计算**：根据出生日期时间自动计算八字
- 💕 **缘分匹配**：基于五行相生相克匹配用户
- 📱 **隐私保护**：手机号仅匹配成功后展示
- 🔑 **手机号登录**：换会话后可用手机号找回报名
- ⏰ **每日报告**：每晚8点推送匹配结果

## 安装

```bash
clawhub install loveclaw
# 或
/skills install loveclaw
```

安装后定时任务自动配置完成。

## 匹配规则

### 匹配条件

1. **同城必须**：双方必须在同一城市
2. **双向确认**：A喜欢B，B也必须喜欢A
3. **匹配阈值**：双向分数都 ≥70% 才算成功

### 匹配分数

- 五行相生：100%
- 五行比和：80%
- 五行相克：50%

### 选择逻辑

- 每日只匹配1人（分数最高者）
- 同城用户优先匹配
- 异地用户不参与匹配

## 每日流程

- **19:50** — 系统自动执行每日匹配
- **20:00** — 推送匹配报告给用户

### 无匹配通知

如果当日没有同城双向匹配，发送：
> 🌙 今日缘分未到...  
> 命运的齿轮继续转动，请期待月老明日的光临 ✨

## 使用命令

| 命令 | 功能 |
|------|------|
| `开启匹配` / `报名` | 开始报名流程 |
| `手机号登录` | 换会话后用手机号找回报名 |
| `我的档案` | 查看报名信息 |
| `今日匹配` | 查看今日匹配结果 |
| `修改` | 修改报名信息 |
| `取消报名` | 删除个人数据 |

## 完整报名流程

1. 输入「开启匹配」
2. 输入手机号（用于登录和通知）
3. 输入姓名
4. 选择性别
5. 选择偏好性别
6. 输入出生日期（格式：YYYY-MM-DD）
7. 输入出生时辰（24小时制，如 14 代表下午2点）
8. 输入城市
9. 发送照片（或输入「跳过」）
10. 回复「确认」完成报名

## 数据存储

- **本地缓存**：用户数据缓存到 `~/.openclaw/workspace/loveclaw/data/`
- **云端同步**：数据通过 API 与云端同步

## 目录结构

```
loveclaw/
├── SKILL.md           # 技能定义
├── README.md          # 说明文档
├── _meta.json         # 元信息
├── cron-config.json   # 定时任务配置
├── package.json       # 依赖
└── scripts/
    ├── cloud-handler.js  # 会话处理（主入口）
    ├── cloud-data.js     # 云端数据操作
    ├── cloud-cron.js     # 云端定时任务
    ├── api-client.js     # API 客户端
    ├── bazi.js           # 八字计算
    ├── match.js          # 匹配引擎
    ├── data.js           # 数据管理
    ├── local-data.js     # 本地数据
    └── cron.js           # 本地定时任务
```

## 隐私说明

1. 手机号仅匹配成功后对匹配对象可见
2. 照片上传至云端存储
3. 用户可随时输入「取消报名」删除所有数据
